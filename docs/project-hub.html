Quick Setup

  1. Fork or clone the repository.
  2. In GitHub repo settings, enable Pages from branch main and folder /docs.
  3. Run the workflow named Build News Site from the Actions tab.
  4. Verify your page at https://YOUR_USERNAME.github.io/techInsights/.

Local Build

Use this command sequence locally:

pip install -r requirements.txt

python scripts/build.py

open docs/index.html

Notes

  • The AI Insight Snapshot is generated in the browser on demand.
  • If model loading fails, the page falls back to non-AI content without blocking the site.
  • Generated static files live under docs/.
